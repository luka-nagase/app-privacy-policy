2. 取得する情報

本アプリ／本システムは、Instagram Graph API 等を通じて、公開情報を中心に以下の情報を取得する場合があります。

  • 投稿ID
  • 投稿時刻（timestamp）
  • キャプション（投稿本文）
  • ハッシュタグ
  • メディア種別（画像/動画 等）
  • メディアURL（media_url）
  • パーマリンク（permalink）
  • いいね数・コメント数等（取得できる範囲）

※原則として、個人を直接特定するための情報（氏名・住所・電話番号等）を収集することは目的としていません。

3. 利用目的

  • 公共空間・観光・都市に関する行動傾向の分析（研究・統計）
  • 地理的傾向の可視化（例：地点/グリッド集計）
  • 時間帯・曜日・季節性等の分析
  • 研究結果の作成（論文、レポート、発表資料等）

4. 位置情報（ジオタグ）の取り扱い

本システムは、投稿本文・ハッシュタグ等のテキスト情報から地名・施設名等を推定し、外部の地理情報サービス（例：OpenStreetMap等）を用いて緯度経度を付与する場合があります。推定により付与された位置情報は、研究・統計目的で集計・分析に用います。

5. 第三者提供

取得データを、個人が特定されうる形で第三者に提供することはありません。研究成果として公開する場合は、統計的に集計された情報（例：グリッド集計、件数分布、時間帯傾向等）を中心に取り扱い、必要に応じて匿名化・非特定化等の措置を行います。

6. 委託

分析・保管のために、クラウドサービス（例：Google Sheets、Google Drive 等）を利用する場合があります。各サービスはそれぞれのプライバシーポリシーに基づき運用されます。

取得データへのアクセス権管理、保存先のアクセス制限、不要データの削除等、適切な安全管理措置を講じます。

10. 本ポリシーの変更

本ポリシーは、法令・サービス仕様の変更、運用上の必要に応じて改定されることがあります。改定後は、本ページにて公表します。
