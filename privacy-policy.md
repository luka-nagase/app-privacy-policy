# プライバシーポリシー

最終更新日: [2026-03-14]

[サービス名]（以下、「当サービス」）は、Pinterest API を利用する機能の提供にあたり、利用者の情報を適切に取り扱うことを重視しています。本プライバシーポリシーは、当サービスが取得・利用・管理する情報、およびその取扱い方針について定めるものです。

## 1. 基本方針

当サービスは、Pinterest およびエンドユーザーに対して誠実かつ透明であることを重視し、ユーザーに困惑、誤解、または不意の不利益を与えるような情報取得・利用は行いません。

当サービスは、Pinterest の技術文書、利用規約、関連ポリシー、および適用される法令を遵守します。

## 2. 取得する情報

当サービスは、Pinterest API を通じて、当サービスの機能提供に必要な範囲でのみ情報にアクセスします。

取得対象となる情報には、以下が含まれる場合があります。

- Pinterest アカウントに関連する基本情報
- ボード、ピン、広告、キャンペーン、分析情報等のうち、ユーザーが明示的に許可した情報
- API 利用に必要な認証情報またはアクセストークン
- 当サービスの提供および動作確認に必要な技術情報

## 3. 情報の利用目的

当サービスは、取得した情報を以下の目的に限って使用します。

- ユーザーに対する当サービスの提供
- Pinterest API を利用した機能の実行
- キャンペーン分析情報の表示または提供
- 障害対応、保守、セキュリティ確保
- ユーザーサポート対応
- 法令遵守および不正利用防止

当サービスは、上記以外の目的でユーザーのアカウント情報を利用しません。

## 4. 情報の保存に関する方針

当サービスは、**Pinterest のマテリアルを通じてアクセスした情報について、原則として保存を行いません。**  
必要な情報へのアクセスが必要になる都度、Pinterest API を呼び出して取得します。

ただし、以下の情報については、必要に応じて保存する場合があります。

- ユーザーのアカウントに関してアクセスした**キャンペーン分析情報**
- セッション維持または認証処理のために必要な最小限の情報
- 法令上またはセキュリティ上必要なログ情報

保存する場合も、必要最小限の範囲・期間に限定し、適切な安全管理措置を講じます。

## 5. 認証およびアカウントアクセス

当サービスは、ユーザー本人の明示的な許可なく、Pinterest アカウントへアクセスしません。

当サービスは、以下の行為を行いません。

- ユーザーの Pinterest ログイン資格情報（ID、パスワード等）の要求または収集
- ユーザーのログイン資格情報を利用したログイン
- 他人のアカウントへのアクセス
- 他人の代わりに操作を行う行為
- 許可なくアクセストークンその他の認証情報を用いてアカウントへアクセスする行為

当サービスが使用する認証は、Pinterest が提供する正規の認可フローに基づいて行われます。

## 6. 情報の組み合わせ禁止

当サービスは、ユーザーの Pinterest アカウント情報を、以下と組み合わせません。

- 別のユーザーのアカウント情報
- 他のサービスから取得した情報
- 本来の提供目的を超える外部データ

当サービスは、ユーザーごとの情報を、当該ユーザーへのサービス提供に必要な範囲でのみ取り扱います。

## 7. 第三者提供・共有・販売

当サービスは、Pinterest API から取得した情報を、**第三者に共有または販売しません。**

特に、以下のような第三者との共有・販売は行いません。

- 他の広告サービス
- マーケティング事業者
- データブローカー
- 分析データの再販売を目的とする第三者

ただし、以下の場合を除きます。

- ユーザー本人が保有する別サービス上のアカウントとの連携に必要な場合
- 法令に基づき開示が必要な場合
- ユーザーの明示的な同意がある場合

## 8. API アクセス資格情報の管理

当サービスは、API アクセス資格情報を機密情報として適切に管理します。

- API キー、クライアントシークレット、アクセストークン等は安全に保護します
- 他のユーザーの API 資格情報を使用しません
- 他のユーザーに当サービスの API 資格情報の使用を許可しません
- 不正アクセス、漏えい、改ざん、紛失を防止するため、合理的な安全管理措置を講じます

## 9. BYOK（Bring Your Own Key）への対応

当サービスが BYOK（Bring Your Own Key）に対応する場合、エンドユーザーは自身の API アクセス認証情報を利用して当サービスを使用できます。

この場合、当サービスは以下の条件を満たすものとします。

- エンドユーザーの認証情報は、可能な限り**ローカル環境に保存**されます
- サーバー側で不要に保存しません
- Pinterest ユーザーのパスワード、セッションクッキー等は要求しません
- その他、Pinterest の関連ポリシーおよびガイドラインを遵守します

## 10. ポリシー回避行為の禁止

当サービスは、Pinterest のポリシー執行システムの回避を試みません。

たとえば、Pinterest によりアプリの削除、停止、制限等が行われた場合に、別アカウントや別名称を用いて、同一または実質的に同一のアプリを再登録・再接続する行為は行いません。

## 11. セキュリティ

当サービスは、取得した情報および認証情報を保護するため、合理的かつ適切な技術的・組織的安全管理措置を講じます。

これには以下が含まれる場合があります。

- 通信の暗号化
- アクセス制御
- 資格情報の安全な保管
- ログ管理
- 不正利用の監視
- 必要最小限の権限管理

ただし、インターネット上の通信または電子保存手段は、完全な安全性を保証するものではありません。

## 12. 法令遵守

当サービスは、適用されるすべての法令、規則、ガイドラインに従って個人情報および関連データを取り扱います。

ユーザーは、当サービスを利用することにより、本ポリシーに記載された範囲での情報取扱いについて理解したものとみなされます。

## 13. プライバシーポリシーの変更

当サービスは、法令改正、Pinterest のポリシー変更、またはサービス内容の変更等に応じて、本プライバシーポリシーを改定することがあります。

重要な変更がある場合は、当サービス上または適切な方法により通知します。
